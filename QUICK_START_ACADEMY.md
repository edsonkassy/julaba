## 🔄 Changer de profil rapidement

```
1. Clic sur Dev Mode (top-right)
   ↓
2. Sélection d'un autre profil
   ↓
3. L'app change automatiquement de contexte
   ↓
4. Navigation vers la page d'accueil du nouveau rôle
   ↓
5. Testez Academy avec les nouvelles formations !
```
